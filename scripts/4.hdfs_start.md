# 4.HDFS Start

## Start HDFS
```
shstart-all.sh
```

## Check Daemons
```
jps
```

* From NameNode:  ```NameNode```  ```ResourceManager```  ```JobHistoryServer```  ```Jps```
* From SecondaryNameNode:  ```SecondaryNameNode```  ```Jps```
* From Each DataNode:  ```DataNode```  ```NodeManager```  ```Jps```
